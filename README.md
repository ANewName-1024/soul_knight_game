# Soul Knight Flutter Clone

## Features
- 虚拟摇杆 + 技能键（移动端）
- WASD移动 + X切换武器 + Q/W/E/R技能（PC）
- 6种武器：手枪/霰弹枪/步枪/狙击枪/魔法杖/导弹
- 3种敌人：追击者/射手/Boss
- 地牢房间生成系统 + 粒子特效
- 游戏界面（开始/暂停/结算）+ SharedPreferences存档

## Build
flutter build apk --debug

## Tech
Flutter 3.41.6 + Flame 1.37.0 + SharedPreferences
