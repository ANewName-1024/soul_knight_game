# Changelog

## v1.0.0 (2026-05-29)

### Features
- Phase 1-8 Complete Release
- Virtual joystick (mobile) + WASD/keys (PC)
- 6 weapon types with switch system
- 3 enemy types: Chaser, Shooter, Boss
- Dungeon room generation system
- Particle effects (hit/death/magic explosion)
- Game screens (Start/Pause/GameOver/Victory)
- SharedPreferences save system (progress, unlocks, settings)

### Tech
- Flutter 3.41.6 + Flame 1.37.0
