# Soul Knight Clone — 游戏设计方案

## 文档信息

- **项目**：Soul Knight Clone（元气骑士类地牢射击游戏）
- **类型**：2D 俯视角射击 + Roguelike + 动作冒险
- **目标**：Android 移动端
- **引擎**：Flutter + Flame 1.37.0

---

## 第一部分：核心设计

### 1.1 游戏愿景

一款移植自经典手游「元气骑士」的复刻作品，保留其核心玩法：
- 俯视角 2D 射击
- 程序化生成的地牢关卡
- 丰富的武器与技能组合
- 持续的角色成长与roguelike体验

### 1.2 目标用户

- 休闲动作游戏玩家
- 地牢探险爱好者
- 移动端轻量级玩家

---

## 第二部分：游戏循环

### 2.1 核心循环图

```
┌─────────────────────────────────────────────────────────────────┐
│  进入关卡 → 探索地牢 → 击败敌人 → 获取奖励           │
│       ↑                                    │       │
│       └──── 死亡/退出 ←─────────────────────┘       │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 单局流程

1. **开始界面** — 选择角色/武器，进入关卡
2. **地牢探险** — 房间依次解锁，击退敌人
3. **Boss战** — 每大关卡结尾的强力敌人
4. **结算** — 统计表现，掉落奖励
5. **成长** — 局外天赋/永久属性提升

### 2.3 每日循环

- **体力系统** — 每日恢复体力次数
- **签到奖励** — 连续登录获取资源

### 4.2 武器属性


|------|---------|---------|---------|
| 1 | 1 | 3 | 1 |

| 3 | 3 | 5 | 3 |
| N | N | 3+N | N |
